# 🎬 OTT Platform Analytics – StreamFlix

## Overview

This project analyzes OTT platform viewership data using PostgreSQL to uncover insights related to user engagement, content performance, regional trends, and subscription revenue.

## Dataset

The dataset contains OTT platform viewing records, including user information, regions, shows, genres, watch hours, subscription revenue, and monthly activity data.

## Business Objectives

* Analyze user watch behavior
* Identify top-performing shows
* Measure regional performance
* Track subscription revenue trends
* Generate content rankings
* Evaluate monthly platform growth

## 📑 Project Documentation

This repository includes a detailed **Solved Tasks PDF** containing all assessment questions, SQL solutions, output screenshots, and business insights.

The document serves as a step-by-step walkthrough of the analysis process and provides complete visibility into how each business requirement was solved using PostgreSQL.


## SQL Concepts Used

* CTEs (Common Table Expressions)
* Views
* Aggregate Functions
* Window Functions
* RANK()
* DENSE_RANK()
* LAG()
* LEAD()
* GROUP BY

## Key Analysis Performed

* Users above average watch hours
* Regional watch activity analysis
* Show popularity rankings
* Subscription revenue ranking
* Monthly revenue trend analysis
* Top performing regions
* Most watched shows by region

## Key Insights

* April generated the highest watch hours and subscription revenue.
* Stranger Things was the most watched show on the platform.
* East region delivered the highest watch hours and subscription revenue.

## Tools Used

* PostgreSQL
* SQL
* pgAdmin

## Project Outcome

Developed advanced SQL queries to transform raw OTT platform data into actionable business insights using ranking, trend analysis, and revenue reporting techniques.
